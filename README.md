# 街にある木を市民が推して見守るサービス「ふぁぼツリー」 
## 技術スタック 
JavaScript(ライブラリにLeafletを使用) / Firebase / GitHub
## 機能概要
東京都内の街路樹を地図上で検索・閲覧でき、樹木毎に写真やコメントを投稿・共有できる Web アプリです。
幹周登録など市民参加型の機能も備え、市民は情報共有を、樹木管理者は点検・保守に活用できる仕組みを設計しました。
